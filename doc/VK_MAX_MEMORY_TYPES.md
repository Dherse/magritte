# C Specifications
[`MAX_MEMORY_TYPES`] is the length of an array of [`MemoryType`]
structures describing memory types, as returned in
[`PhysicalDeviceMemoryProperties`]::memoryTypes.
```c
#define VK_MAX_MEMORY_TYPES               32U
```
